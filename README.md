# Connect Four
Full stack implementation of the Connect Four Game.


## Run

### Backend

#### Database

```bash
# start mongodb
mongo
```

#### REST api server

```bash
# change to the frontend directory
cd backend

# install dependencies
npm install

# compile typescript source code
npm run compile

# launch the server
npm start
```

### Frontend

```bash
# change to the frontend directory
cd frontend

# install dependencies
npm install

# launch the web app
ng serve --open
```
